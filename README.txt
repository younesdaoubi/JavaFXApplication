# Automated Warehouse

## Description
Automated Warehouse est une application permettant de gérer un entrepôt automatisé. Il reçoit des livraisons de composants de base (batterie, moteur électrique ou capteur de mouvement) et assemble des produits finis à partir de ces composants.

Par exemple, à partir d'une batterie et d'un capteur de mouvement, le système peut créer un robot suiveur. De même, en combinant d'autres composants, il est possible de fabriquer divers produits.

Le projet utilise une stratégie de conception automatisée basée sur les Design Patterns, garantissant flexibilité et maintenabilité du code.
